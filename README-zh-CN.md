# 数字信号处理 Course Work

## 简介
每帧进行时域与频域处理，帧与帧之间提取时间序列特征。

| 时域特征           | SVM    |        |
| 时域特征           | 决策树 |        |

| 梅尔倒谱系数(MFCC) | 决策树 |        |
